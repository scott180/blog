# pagefile.sys有几十个G，太大怎么办？

# 说明

pagefile.sys是虚拟内存文件，目地是使用一部分硬盘空间来充当内存使用。当一个程序请求的内存空间大于物理内存时，就需要pagefile.sys文件来提供较大的虚拟内存，从而满足程序对大内存的需求。
一般来说，pagefile.sys是不可以删除的，但是我们可以将该文件从一个磁盘转移到另外的磁盘中。

参考  pagefile.sys怎么删除 (opens new window) pagefiles.sys文件的调整与删除 (opens new window)
网址  blog (opens new window) gitlab (opens new window)

# 文件移动

将pagefile.sys从C盘移到E盘。

1、打开控制面板的系统设置。   图解 (opens new window)

2、选中性能选项-高级-更改。   图解 (opens new window)

3、选中c盘，选择无分页文件，点击设置。注意：需要取消选中“自动管理所有驱动器的分页文件大小”。   图解 (opens new window)

4、选择E盘，选择系统管理的大小，点击设置，确定。   图解 (opens new window)

5、重启计算机。   图解 (opens new window)

# 查看文件大小

在工具-文件夹选项取消选中“隐藏受保护的操作系统文件（推荐）”，即可查看pagefile.sys文件大小。
选中“隐藏受保护的操作系统文件（推荐）”，即恢复正常。

1、选择工具-文件夹选项   图解 (opens new window)

2、查看-取消选中“隐藏受保护的操作系统文件（推荐）”，警告弹窗选择“是”   图解 (opens new window)

3、确定   图解 (opens new window)

4、查看pagefile.sys文件大小   图解 (opens new window)
